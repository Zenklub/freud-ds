## Pré-requisitos

### Libs

É necessário instalar a lib de `web-components` para o correto funcionamento do componente:

```bash
npm i @freud-ds/web-components
```

## <a id="importing"></a>Import

Importe o `FreudTooltipModule` em cada módulo que for utilizar o `FreudTooltipComponent` em seu projeto:

```ts
import { FreudTooltipModule } from '@freud-ds/web-components';
```

## Tooltip

Para utilizar o componente, basta utilizar a tag `<span freud-tooltip>` passando os parâmetros necessários.
