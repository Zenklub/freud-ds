## Getting Started

É necessário o componente `freud-toast` definido no html e o serviço `FreudMessageService` para a inclusão das mensagens

Toast oferece suporte a várias posições em que o padrão é "top-right".

```html
<freud-toast [position]="'top-center'" url="https://google.com"></freud-toast>
```

```ts
import { FreudMessageService, FreudMessage } from "@freud-ds/web-components";

export class ComponentDemo {
  constructor(private messageService: FreudMessageService) {}

  confirm() {
    const message: FreudMessage = {
      severity: "success",
      summary: "Service Message",
      detail: "Via MessageService",
    };
    this.messageService.add(message);
  }
}
```
