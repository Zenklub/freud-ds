## API

As mensagens a serem exibidas podem ser definidas usando a propriedade `messages` sendo uma lista de `FreudMessage` ou usando o serviço `FreudMessageService`.

### Utilizando via componente

```ts

show() {
    this.msgs.push({severity:'info', summary:'Info Message', detail:'Freud DS'});
}

hide() {
    this.msgs = [];
}

```

```html
<freud-confirm-dialog [messages]="messages"></freud-confirm-dialog>
```

### Utilizando via serviço

É necessário o componente `freud-confirm-dialog` definido no html

```html
<freud-confirm-dialog></freud-confirm-dialog>
```

```ts
import { FreudMessageService } from '@freud-ds/web-components';

export class ComponentDemo {
  constructor(private messageService: FreudMessageService) {}

  confirm() {
    this.messageService.add({
      severity: 'success',
      summary: 'Service Message',
      detail: 'Via MessageService',
      icon: 'freud-icon freud-icon-check-circle',
    });
  }
}
```

### Severities

Aqui estão os valores possíveis para a gravidade de uma mensagem.

- success
- info
- warn
- error
