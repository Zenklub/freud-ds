## Getting Started
FreudTabView requer uma coleção de templates `freud-tab-panel`.
```html
    <freud-tab-view
      [activeIndex]="activeIndex"
      [controlClose]="true"
      [scrollable]="false"
      [bgColor]="false"
      (onClose)="handleClose($event)">
        <freud-tab-panel header="Header 1">
            Content 1
        </freud-tab-panel>
        <freud-tab-panel header="Header 2" [disabled]="true">
            Content 2
        </freud-tab-panel>
        <freud-tab-panel header="Header 3">
            Content 3
        </freud-tab-panel>
    </freud-tab-view>
```
