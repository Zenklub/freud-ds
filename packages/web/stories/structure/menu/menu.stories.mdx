## Getting Started

FreudMenu requer uma coleção de itens de menu, para isso utilize a interface `FreudMenuItem`

```ts
import { FreudMenuItem } from '@freud-ds/web-components';

items: FreudMenuItem[] = [{
  label: 'Options',
  items: [{
    label: 'Update',
    icon: 'freud-icon freud-icon-refresh',
    command: () => {/* Comando */ }
  },
    {
      label: 'Delete',
      icon: 'freud-icon freud-icon-times'
    }
  ]}
]
```

```html
<freud-menu [items]="items" [popup]="false"> </freud-menu>
```

## Popup Mode

O menu é embutido por padrão, o modo pop-up também é compatível com a ativação da propriedade pop-up e o método de alternância de chamada passando o evento do elemento âncora.

```html
<freud-menu #menu [popup]="true" [items]="items"></freud-menu>
<freud-button (click)="menu.reference.toggle($event)"
  >Clique para mostrar menu!</freud-button
>
```
