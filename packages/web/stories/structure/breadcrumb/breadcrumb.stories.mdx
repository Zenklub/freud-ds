## Getting Started
Breadcrumb requer uma coleção de itens de menu, para isso utilize a interface `FreudMenuItem`
```ts
import { FreudMenuItem } from '@freud-ds/web-components';

items: FreudMenuItem[] = [
  {label: 'Item1'},
  {label: 'Item2'},
  {label: 'Item3'},
  {label: 'Item4'}
]

home: FreudMenuItem = {
  icon: 'freud-icon freud-icon-home'
}
```
```html
    <freud-breadcrumb
    [bgColor]="false"
    [items]="items"
    [home]="home"
    (onItemClick)="aoClicarNoItem($event)">
</freud-breadcrumb>
```
