## Getting Started

O painel é definido com o elemento freud-panel.

```html
<freud-panel header="Title"> Content </freud-panel>
```

### Header and Footer Content

As seções de cabeçalho e rodapé podem ser personalizadas usando modelos de `header` e `footer`.

```html
<freud-panel>
  <ng-template freudTemplateHeader> Header content here </ng-template>
  Body Content
  <ng-template freudTemplatefooter> Footer content here </ng-template>
</freud-panel>
```

### Custom Icons

Ícones adicionais podem ser colocados na seção de cabeçalho do painel usando o template `freudTemplateIcons`.
Para uma aparência unificada, é recomendável adicionar a classe .p-panel-header-icon aos seus ícones.

```html
<freud-panel>
  <freud-button
    pButton
    class="p-panel-header-icon p-link"
    (click)="menu.toggle($event)"
  >
    <span class="freud-icon freud-icon-cog"></span>
  </freud-button>
</freud-panel>
```
