## Getting Started
Menubar requer uma coleção de itens de menu, para isso utilize a interface `FreudMenuItem`
```ts
import { FreudMenuItem } from '@freud-ds/web-components';

items: FreudMenuItem[] = [
  {label: 'Item1'},
  {label: 'Item2'},
  {label: 'Item3'},
  {label: 'Item4'}
]
```
```html
      <freud-menubar
          [startImgAlt]="'Home'"
          [startImgSrc]="'assets/img/zenklub.svg'"
          [startImgHref]="'https://zenklub.com.br'"
          [items]="items"
          [bgColor]="bgColor">
</freud-menubar>
```
