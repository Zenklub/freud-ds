## Getting Started
FreudTabMenu requer uma coleção de itens de menu, para isso utilize a interface `FreudMenuItem`
```ts
import { FreudMenuItem } from '@freud-ds/web-components';

items: FreudMenuItem[] = [
  {label: 'Home', command: () => { /* Ação quando for clicado na opção */}},
  {label: 'Calendar', command: () => { /* Ação quando for clicado na opção */}},
  {label: 'Edit', command: () => { /* Ação quando for clicado na opção */}},
  {label: 'Documentation', command: () => { /* Ação quando for clicado na opção */}},
  {label: 'Settings', command: () => { /* Ação quando for clicado na opção */}}
]
```
```html
      <freud-tab-menu
          [items]="items"
          [activeItem]="items[0]"
          [bgColor]="bgColor">
</freud-tab-menu>
```
