## Getting Started
FreudSlideMenu requer uma coleção de itens de menu, para isso utilize a interface `FreudMenuItem`
```ts
import { FreudMenuItem } from '@freud-ds/web-components';

items: FreudMenuItem[] = [{
  label: 'Options',
  items: [{
    label: 'Update',
    icon: 'freud-icon freud-icon-refresh',
    command: () => {/* Comando */ }
  },
  {
    label: 'Delete',
    icon: 'freud-icon freud-icon-times'
  }
  ]}
]
```
```html
      <freud-slide-menu [items]="items"></freud-slide-menu>
```
