## Getting Started
FreudDialog é definido usando a tag freud-dialog e exibido/oculto pela propriedade `visible`.

Utilize o template `freudTemplatefooter` para exibir o footer do modal
```html
<freud-dialog header="Header" [(visible)]="visible">
  <p>Conteudo do modal</p>

  <ng-container freudTemplatefooter>
    <freud-button color="secondary" (click)="visible=false">Cancelar</freud-button>
    <freud-button (click)="visible=false">Ok</freud-button>
  </ng-container>

</freud-dialog>
```
