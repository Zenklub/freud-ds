## Libs

É necessário instalar algumas libs do Freud DS para utilizar este componente. Para instalar, rode os comandos abaixo no terminal do seu projeto:

### Web Components

```ts
npm i @freud-ds/web-components --legacy-peer-deps
```

### Icons

```ts
npm i @freud-ds/icons --legacy-peer-deps
```

## <a id="importing"></a>Import
Importe o FreudIconComponent e o FreudIconModule no seu arquivo. Utilizando os comandos:
```ts
import { FreudIconComponent, FreudIconModule } from '@freud-ds/web-components'; 
```

## Icon
Para utilizar o componente, basta utilizar a tag ```<freud-icon>```passando o nome do ícone (e, caso seja necessário, 
o tamanho) como parâmetro. Para ver a lista de ícones disponíveis, 
[clique aqui](https://zenklub.github.io/freud-ds/iframe.html?viewMode=docs&id=freud-ds-web-components-icons--storyname-hidden#a-idlistalisting).

### Usage

```ts
<freud-icon icon="[icon name]"></freud-icon>
```

```ts
<freud-icon icon="[icon name]" size="sm"></freud-icon>
```
