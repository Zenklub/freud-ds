## <a id="importing"></a>Import
### Estilo
Importe o arquivo de estilo dos ícones:

#### CSS Global
O arquivo de estilo dos ícones será importado globalmente, ou seja, em toda a aplicação. Para isso, coloque esse código 
no seu arquivo ``.css`` global:
```css
@import '@freud-ds/icons/style/style.css';
```

## Icon
Para utilizar o componente, basta utilizar a tag ``<i>`` ou ``<span>`` passando a classe com o nome do ícone (``freud-icon freud-icon-[icon-name]``)
como parâmetro.
