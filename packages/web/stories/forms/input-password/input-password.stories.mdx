## Getting Started
O componente é definido usando a tag `freud-input-password` e a associação de valor bidirecional é habilitada com a diretiva ngModel padrão.

### Customization

O componente de senha usa expressões regulares para senhas médias e fortes com os seguintes padrões.

#### Medium
^(((?=.*[a-z])(?=.*[A-Z]))|((?=.*[a-z])(?=.*[0-9]))|((?=.*[A-Z])(?=.*[0-9])))(?=.{6,}).

- Pelo menos uma letra minúscula
- Pelo menos uma letra maiúscula ou um número
- Mínimo 6 caracteres

#### Strong
^(?=.*[a-z])(?=.*[A-Z])(?=.*[0-9])(?=.{8,})

- Pelo menos uma letra minúscula
- Pelo menos uma maiúscula
- Pelo menos um numérico
- Mínimo 8 caracteres

É possível definir suas próprias verificações com as propriedades `mediumRegex` e `strongRegex`.

### Templating
2 modelos estão incluídos para personalizar a sobreposição. Estes são `header` e `footer`. Observe que o conteúdo substitui o medidor padrão.
```
<freud-input-password [(ngModel)]="value4">

    // É possível incluir um texto acima da barra de dificuldade da senha
    <ng-container freudTemplateHeader>
      <strong>Pick a password</strong>
    </ng-container>

    // É possível incluir um footer
    <ng-container freudTemplatefooter>
        <div>
          <hr>
          <p class="p-mt-2">Suggestions</p>
          <ul class="p-pl-2 p-ml-2 p-mt-0" style="line-height: 1.5">
              <li>At least one lowercase</li>
              <li>At least one uppercase</li>
              <li>At least one numeric</li>
              <li>Minimum 8 characters</li>
          </ul>
        </div>
    </ng-container>
</freud-input-password>
```
