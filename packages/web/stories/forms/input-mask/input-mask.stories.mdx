## Getting Started

### Mask
O formato da máscara pode ser uma combinação das seguintes definições integradas.

- a - Alfa caracteres (padrão: A-Z,a-z)
- 9 - Números (0-9)
- * - Alfa númerico caracteres (A-Z,a-z,0-9)

```html
<freud-input-mask [(ngModel)]="val" mask="a*-999-a999"></freud-input-mask>
```

Você também pode definir seu próprio padrão regex para caracteres alfa

```html
<freud-input-mask formControlName="username" mask="(999) 999-9999? x99999"></freud-input-mask>
```
