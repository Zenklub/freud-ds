## Getting Started

O modo múltiplo é ativado por padrão, a propriedade ngModel refere-se a uma matriz para vincular os valores selecionados.

### Model Driven Forms

A caixa de seleção também pode ser usada em um formulário orientado a modelo.
Neste caso, devido a um problema no Angular, vincule a instância formControl em vez de usar formControlName.

```html<!-- Wrong -->
<freud-checkbox formControlName="cities"></freud-checkbox>

<!-- Correct -->
<freud-checkbox [formControl]="myFormGroup.controls['cities']"></freud-checkbox>
```
