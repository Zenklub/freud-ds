## Getting Started

A ligação de valor bidirecional é definida usando a diretiva `ngModel` padrão.
Utilize a propriedade `radioValue` para definir o valor do Radio

```html
<freud-radio-button
  [(ngModel)]="value"
  [name]="'radio-name'"
  [radioValue]="'item2'"
  [bgColor]="false"
>
</freud-radio-button>
```
