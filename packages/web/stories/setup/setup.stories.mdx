# Freud DS - Comece aqui

O Freud DS é o Design System do Zenklub.

Esta documentação contempla a biblioteca de componentes para Angular, a `@freud-ds/web-components`.

## Pré-requisitos

### Versão 3.x

- Angular 16

### Versão 2.x

- Angular 12

## Utilização

### 1 - Instale as seguintes `peer dependencies`

```json
{
  "@angular/animations": "16.2.x",
  "@angular/cdk": "16.2.x",
  "@angular/common": "16.2.x",
  "@angular/core": "16.2.x",
  "@angular/forms": "16.2.x",
  "@freud-ds/icons": "2.x",
  "@freud-ds/tokens": "2.x"
}
```

### 2 - Instale a lib

```bash
npm i @freud-ds/web-components
```

### 2 - Importe os estilos da lib

No arquivo `angular.json` importe os estilos da lib

```json
...
"styles": [
  "node_modules/@freud-ds/web-components/scss/styles.scss",
  "node_modules/@freud-ds/web-components/scss/typography.scss"
],
...
```

### 3 - Importe o modulo do componente a ser utilizado conforme documentação no storybook

Exemplo: `freud-button`

```ts
// some-component.module.ts

import { FreudButtonModule } from '@freud-ds/web-components';

@NgModule({
  imports: [
    ...
    FreudButtonModule
  ]
})
```

```html
<!-- some-component.component.html -->

<freud-button label="Click me!"></freud-button>
```

Veja a documentação de cada componente da `@freud-ds/web-components` para todas opções disponíveis.
