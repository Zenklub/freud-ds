## Getting Started

Defina a lista de imagens a serem exibidas utilizando a interface `GalleryPhoto`.

```ts

import { GalleryPhoto } from '@freud-ds/web-components';

images: GalleryPhoto[] = [
    {
        "type": "image"
        "previewImageSrc": "demo/images/galleria/galleria5.jpg",
        "thumbnailImageSrc": "demo/images/galleria/galleria5s.jpg",
        "alt": "Description for Image 5",
        "title": "Title 5"
    },
    {
        "type": "video"
        "previewImageSrc": "https://www.youtube.com/embed/AurVL6n56wg",
        "thumbnailImageSrc": "https://www.youtube.com/embed/AurVL6n56wg",
        "alt": "Description for Image 5",
        "title": "Title 5"
    }
]

```

No html use as propriedades `showThumbnails` e `bgColor` para definir a exibição dos thumbnails e cor do background

```html
<freud-gallery
  [images]="images"
  [showThumbnails]="showThumbnails"
  [bgColor]="bgColor"
  [(activeIndex)]="activeIndex"
>
</freud-gallery>
```
