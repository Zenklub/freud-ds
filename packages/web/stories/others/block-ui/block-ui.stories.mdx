## Getting Started
BlockUI é controlado usando a propriedade bloqueada e o componente a ser bloqueado é definido como destino.
Se o destino não for fornecido, o próprio documento será selecionado como o destino do bloco.
```html
<freud-block-ui [blocked]="blocked"></freud-block-ui>
```
