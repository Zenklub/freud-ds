## Getting Started
Captcha é usado com um siteKey e um retorno de chamada para verificar a resposta.
```html
<freud-captcha siteKey="YOUR_SITE_KEY" (onResponse)="showResponse($event)"></freud-captcha>
```

Além disso, inclua o recurso de widget captcha em sua página.
```html
<script src="https://www.google.com/recaptcha/api.js?render=explicit&onload=initRecaptcha" async defer></script>
```
O nome do callback global é initRecaptcha por padrão e pode ser alterado usando a propriedade initCallback.
```html
<script src="https://www.google.com/recaptcha/api.js?render=explicit&onload=loadCaptcha" async defer></script>
```
```html
<freud-captcha siteKey="YOUR_SITE_KEY" (onResponse)="showResponse($event)" initCallback="loadCaptcha"></freud-captcha>
```

### Verification
Para garantir que um token de resposta seja válido, a verificação em relação à recaptcha api precisa ser feita no back-end.
Leia mais na documentação oficial: https://developers.google.com/recaptcha/docs/verify
```ts
showResponse(response) {
    //Chamada para o backend verificar o token
}

```
