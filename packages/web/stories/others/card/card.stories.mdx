## Getting Started

O card é definido com o elemento `freud-card`.

```html
<freud-card header="Simple card"> Content </freud-card>
```

### Templating

2 modelos estão incluídos para personalizar a sobreposição. Estes são `header` e `footer`. Veja o exemplo `Advanced` abaixo

```html
<freud-card header="Advanced card">
  <ng-container freudTemplateHeader>
    <img alt="Card" src="assets/img/img.png" style="height: 150px" />
  </ng-container>

  Card Content
  <ng-container freudTemplatefooter>
    <div style="display: flex; gap: 4px">
      <freud-button>Button</freud-button>
      <freud-button [color]="'secondary'">Button</freud-button>
    </div>
  </ng-container>
</freud-card>
```
